Payment Gateway for Telcell-ը միջազգային WooCommerce հարթակի պլագին է, որը հնարավորություն է ստեղծում Ձեր կայքը հեշտությամբ ինտեգրել Telcell վճարային համակարգին՝ Telcell Wallet հավելվածի միջոցով վճարումներ ընդունելու նպատակով։



== Նկարագրություն ==
 
Պլագինը նախատեսված է ՀՀ տարածքում գործող տնտեսավարող սուբյեկտների համար և հնարավորություն ունի վճարումներ ընդունել միայն ՀՀ դրամով։
Այն պատրաստված է WooCommerce հարթակի տեխնիկական բնութագրին համապատասխան՝ Telcell համակարգի տեխնիկական բնութագրի հիման վրա։

Նախքան պլագինի օգտագործումը անհրաժեշտ է կապ հաստատել «Թել-Սել» ՓԲԸ-ի հետ՝ 060272227 հեռախոսահամարով կամ 10xbusiness@telcell.am էլ․հասցեով, համապատասխան պայմանագիր կնքելու և կարգավորումներ ստանալու համար:

Telcell Wallet հավելվածի (էլեկտրոնային դրամապանակի) միջոցով վճարման գործընթացը լիովին ավտոմատացված է։

Պլագինի աշխատանքային նկարագիր

Պլագինի ինտեգրումից և համապատասխան կարգավորումներից հետո Ձեր կայքում հայտնվում է Telcell Wallet վճարամիջոցի կոճակը։ Երբ օգտատերն ընտրում է այդ վճարամիջոցը, նրա Telcell Wallet էլեկտրոնային դրամապանակում հայտնվում է վճարման հաշիվը, որը օգտատերը հաստատում է կամ չեղարկում, ինչը արտացոլվում է կայքում՝ վճարման կարգավիճակի ամրագրմամբ։ Ձեր կայքի կառավարման վահանակում, ինչպես նաև «Թել-Սել» ՓԲԸ տրամադրած համակարգում կարող եք դիտել կատարված գործարքների կարգավիճակը, հետևել վճարումների պատմությանը։

Պլագինի օգտագործման տեխնիկական պայմաններ

•	Ունենալ կայք՝ պատրաստված WordPress համակարգով,
•	Կայքում ինտեգրել WooCommerce հարթակը,
•	Կայքը համապատասխանեցնել պլագինի նվազագույն պահանջներին,
•	Կայքում ներքին հղումները տեղադրել ճշգրիտ։
 
Պլագինի հնարավորությունները (տես՝ նկ․ 1)

1. Ձեզ հնարավորություն է տրվում խմբագրել «Անվանում» և «Նկարագրություն» դաշտերը։
2. Պլագինը հաջողությամբ թեստավորվել է, և այն համատեղելի է թարգմանությունների միջազգային հայտնի WPML պլագինի հետ, որը հնարավորություն է ընձեռում այս պլագինն օգտագործել բազմալեզու կայքերի համար։
3. Առկա է կառավարման եռալեզու վահանակ։
4. Պլագինն ունի՝ 
•	թեստի հնարավորություն (մանրամասների համար անհրաժեշտ է կապ հաստատել «Թել-Սել» ՓԲԸ-ի հետ),
•	տեխնիկական լոգավորման հնարավորություն,
•	մեկ հպման տրամաբանությամբ թարմացման հնարավորություն,
•	զամբյուղի պարունակության պահպանման հնարավորություն։

Ուշադրություն

Հարգելի գործընկեր, տեղեկացնում ենք, որ հարթակից օգտվելիս Ձեր կայքի անվանումը (domain), կայքի էլ․ հասցեն և IP հասցեն հասանելի է դառնալու մեր ընկերությանը՝ ընկերության հեղինակային իրավունքների պաշպանության նկատառումներից ելնելով, ինչպես նաև «Թել-Սել» ՓԲԸ-ին։


 
== Ինտեգրում ==
 
Հավելվածի ինտեգրմումը կարելի է կատարել հետևյալ տարբերակներով
 
Տարբերակ 1 (FTP)
 
1. Ներբեռնել `Payment-Gateway-for-Telcell.zip`-ը `/wp-content/plugins/` տարածք
2. Կատարել ակտիվացում WordPress-ի կառավարման վահանակի “Plugins” բաժնում
3. Կատարել համապատասխան կարգավորումները և մուտքագրել ԹԵԼ-ՍԵԼ ՓԲԸ-ի կողմից տրամադրված տեղեկատվությունը



Տարբերակ 2

1․Wordpress կառավարման վահանակում մուտք գործել “Plugins” բաժին։
2․Սեղմել «Ավելացնել նոր հավելված» կոճակը (“Add new plugin”)  և ներբեռնել `Payment-Gateway-for-Telcell.zip` ֆայլը։
3. Կատարել համապատասխան կարգավորումները և մուտքագրել ԹԵԼ-ՍԵԼ ՓԲԸ-ի կողմից տրամադրված տեղեկատվությունը։
